## üìê Responsive Breakpoints Used

### Mobile First Approach
- **Base (Mobile):** < 768px
- **md (Tablet):** ‚â• 768px
- **lg (Desktop):** ‚â• 1024px

### Spacing Scale
**Padding (py):**
- Mobile: `py-12` (3rem / 48px)
- Tablet: `py-20` (5rem / 80px)
- Desktop: `py-24` (6rem / 96px)

**Margins (mb):**
- Mobile: `mb-10` (2.5rem / 40px)
- Tablet/Desktop: `mb-16` (4rem / 64px)

**Gaps:**
- Mobile: `gap-3` or `gap-4` (0.75rem-1rem)
- Tablet: `gap-4` or `gap-6` (1rem-1.5rem)
- Desktop: `gap-6` or `gap-8` (1.5rem-2rem)

## üéØ Typography Scale

### Headings (H2)
- Mobile: `text-3xl` (1.875rem / 30px)
- Tablet: `text-4xl` (2.25rem / 36px)
- Desktop: `text-5xl` (3rem / 48px)

### Body Text
- Mobile: `text-base` (1rem / 16px)
- Tablet: `text-lg` (1.125rem / 18px)
- Desktop: `text-xl` (1.25rem / 20px)

### Labels
- Mobile: `text-sm` (0.875rem / 14px)
- Tablet: `text-base` (1rem / 16px)
